## 🎬 **Demo Script - Student Experience**

### **Scene 1: Registration**
```
1. Click "Login" → "Register"
2. Fill form:
   - Username: "demo_student"
   - Email: "student@usiu.ac.ke"
   - Password: "demo123"
3. Click "Register"
4. ✅ Shows success message
```

### **Scene 2: Major Recommendation**
```
1. Fill KCSE grades:
   - Math: A
   - English: A-
   - Kiswahili: B+
   - Biology: A
   - Physics: A-
   - Chemistry: B+
   - Humanities: B
   - Tech/Business: A

2. Select Interest: "Science & Technology"
3. Click "Get Recommendation"
4. ✅ Shows: Computer Science, School of Science & Technology, 90% confidence
```

### **Scene 3: View History**
```
1. Click "My History"
2. ✅ Shows recommendation with input summary
3. Click "Delete Account" (don't actually delete!)
```

## 💡 **Demo Tips**

### **Preparation:**
- Have sample KCSE grades ready
- Know admin credentials: `admin@usiu.ac.ke` / `admin123`
- Open browser to `http://localhost:5000`

### **Timing:**
- **Student demo:** 3-4 minutes
- **Admin demo:** 4-5 minutes
- **Total demo:** 8-10 minutes

### **Backup Plan:**
- If app doesn't start: Check Python version (3.8+)
- If database issues: Delete `instance/site.db` and restart
- If login fails: Use admin credentials

### **Questions to Anticipate:**
- **"How does the AI work?"** → Rule-based matching, not ML
- **"Is it accurate?"** → Based on USIU admission criteria
- **"Can students change majors?"** → Yes, unlimited recommendations
- **"What data is stored?"** → Grades, recommendations, user info
- **"Is it secure?"** → Password hashing, role-based access
